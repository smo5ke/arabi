# دليل استخدام لغة عربي (Arabi)

## مقدمة
عربي هي لغة برمجة عربية كاملة ببنية Python-like و أداء C++-competitive.
البنية مبنية على Python — استخدم `:` و indent بدلاً من `{}`.

## التثبيت

```bash
cargo build --release
```

## CLI

### أوامر التشغيل

```bash
# تشغيل ملف
arabi ملفي.txt

# REPL تفاعلي
arabi

# وضع التصحيح (يعرض bytecode)
arabi -d ملفي.txt

# المساعدة
arabi --help

# الإصدار
arabi --version
```

### أوامر REPL

| الأمر | الوصف |
|-------|-------|
| `/مساعدة` | عرض المساعدة |
| `/خروج` | الخروج من REPL |
| `/حالة` | عرض المتغيرات المعرفة |

### أوامر الحزم

```bash
# إنشاء حزمة جديدة
arabi pakg create حزمة_جديدة

# عرض الحزم المثبتة
arabi pakg list

# عرض معلومات حزمة
arabi pakg info حزمة

# حذف حزمة
arabi pakg remove حزمة

# تحديث حزمة
arabi pakg update حزمة
```

### نظام الحزم

الحزم تُخزّن في مجلد `__pakgs__/` في جذر المشروع:

```
__pakgs__/
  حسب/
    __beginning__.txt    # ملف بداية الحزمة
    pakg.json           # ملف التعريف
  وقت/
    __beginning__.txt
    pakg.json
```

ملف `pakg.json`:

```json
{
    "name": "حسب",
    "version": "0.1.0",
    "description": "مكتبة رياضيات",
    "author": "فريق عربي",
    "files": ["__beginning__.txt"]
}
```

الامتدادات المدعومة: `.txt` أو `.عربي`

## الإسناد

```
س = 5                    # إسناد عادي

س += 5                   # رجعي (+, -, *, /, \, %, ^)

# إسناد رمزي (walrus)

```

## الحالات الشرطية

### اذا / اواذا / والا
```
س = 10
اذا س > 9:
    اطبع("أكبر من 9")
اواذا س == 9:
    اطبع("يساوي 9")

```

```

### المقارنات المتسلسلة

```

### مقارنات الهوية

## الدوال

### أساسيات
```
دالة تحية(اسم):
    ارجع "مرحبا " + اسم

اطبع(تحية("أحمد"))    # مرحبا أحمد
```

### معاملات افتراضية
```
دالة potentia(أساس = 2، أُس = 10):
    ارجع أساس ^ أُس

اطبع(potentia())     # 1024
اطبع(potentia(3، 3)) # 27
```

### *args و ^kwargs
```
دالة مختلطة(أ، *الإضافي، ^الكثرة):
    اطبع("أ:", أ)
    اطبع("إضافي:", الإضافي)
    اطبع("كثرة:", الكثرة)

مختلطة(1، 2، 3، 4، اللغة="عربي")
# أ: 1
# إضافي: [2, 3, 4]
# كثرة: {اللغة: عربي}
```

### خطية (lambda)
```
ضعف = خطية س: س * 2
اطبع(ضعف(5))    # 10
```

## الأصناف

### أساسيات
```
صنف مخروط:
    دالة __تهيئة__(هذا، نق = 1):
        هذا.نق = نق

    دالة مساحة(هذا):
        من حساب استورد ط
        ارجع ط * هذا.نق ^ 2

    دالة __نص__(هذا):
        ارجع "مخروط(" + نص(this.نق) + ")"

م = مخروط(5)
اطبع(م.مساحة())    # 78.5...
اطبع(م)            # مخروط(5)
```

### الوراثة
```
صنف حيوان:
    دالة __تهيئة__(هذا، اسم):
        هذا.اسم = اسم

    دالة صوت(هذا):
        ارجع "..."

صنف كلب(حيوان):
    دالة صوت(هذا):
        ارجع "همهمة!"

ك = كلb("ركيزة")
اطبع(ك.صوت())    # همهمة!
```

### __عرض__ و __استدعاء__
```
صنف مضروب:
    دالة __تهيئة__(هذا، ن):
        هذا.ن = ن

    دالة __عرض__(هذا):
        ارجع "مضروب(" + نص(this.ن) + ")"

    دالة __استدعاء__(هذا، ن):
        ارجع مضروب(ن)

م = مضروب(5)
اطبع(م)        # مضروب(5)
اطبع(م(3))     # 6
```

## التغليف (Closures)

```
دالة عداد():
    ع = 0
    دالة زد():
        محلي ع
        ع += 1
        ارجع ع
    ارجع زد

ج = عداد()
اطبع(ج())    # 1
اطبع(ج())    # 2
اطبع(ج())    # 3

# عام (global)
عام عداد_عام = 0
دالة زد_عام():
    عام عداد_عام
    عداد_عام += 1
```

## الأصناف المدمجة (Magic Methods)

| الطريقة | الاستخدام | مثال |
|---------|----------|------|
| `__تهيئة__` | مُنشئ الصنف | `دالة __تهيئة__(هذا، نق)` |
| `__عرض__` | تمثيل رسمي | `اطبع(كائن)` |
| `__نص__` | تحويل لنص | `نص(كائن)` |
| `__استدعاء__` | استدعاء كدالة | `كائن()` |
| `__طول__` | طول الكائن | `طول(كائن)` |
| `__اجمع__` / `__اجمع_ع__` | عملية الجمع (+) | `كائن1 + كائن2` |
| `__طرح__` / `__طرح_ع__` | عملية الطرح (-) | `كائن1 - كائن2` |
| `__ضرب__` / `__ضرب_ع__` | عملية الضرب (*) | `كائن1 * كائن2` |
| `__قسمة__` / `__قسمة_ع__` | عملية القسمة (/) | `كائن1 / كائن2` |
| `__قوة__` / `__قوة_ع__` | عملية القوة (^) | `كائن1 ^ كائن2` |
| `__يساوي__` / `__يساوي_ع__` | مقارنة (==) | `كائن1 == كائن2` |
| `__اصغر__` / `__اصغر_ع__` | مقارنة (<) | `كائن1 < كائن2` |
| `__اكبر__` / `__اكبر_ع__` | مقارنة (>) | `كائن1 > كائن2` |
| `__بادئ__` | بادئ التكرار | `لكل ع في كائن` |
| `__التالي__` | التكرار التالي | `التالي(بادئ)` |
| `__يحتوي__` | فحص الاحتواء | `3 في كائن` |
| `__ادخل__` | دخول السياق | `باستخدام كائن` |
| `__اترك__` | مغادرة السياق | `باستخدام كائن` |

## الزخرفة (Decorators)

الزخرفة تسمح بتعديل الدوال او الاصناف قبل استخدامها. تدعم عربي زخرفتين: `زخرف` و `@`.

### زخرف الدوال
```
زخرف دالة سجل(الدالة):
    دالة ملفوف(*الargs، **ال kwargs):
        اطبع("قبل الاستدعاء")
        النتيجة = الدالة(*الargs، **ال kwargs)
        اطبع("بعد الاستدعاء")
        ارجع النتيجة
    ارجع ملفوف

@سجل
دالة قول(رسالة):
    اطبع(رسالة)

قول("مرحبا")
# قبل الاستدعاء
# مرحبا
# بعد الاستدعاء
```

### زخرف الاصناف
```
زخرف دالة اضف_عرض(الصنف):
    الصنف.عرض_مخصص = دالة(هذا): ارجع "كائن مخصص"
    ارجع الصنف

@اضف_عرض
صنف كائن_مخصص:
    pass
```

## الوصول للخصائص (getattr/setattr)

```
صنف نقطة:
    دالة __تهيئة__(هذا، ص، خ):
        هذا.ص = ص
        هذا.خ = خ

ن = نقطة(3، 4)

# الوصول بال点
اطبع(ن.ص)        # 3

# الوصول الديناميكي
اطبع(خاصية(ن، "ص"))        # 3
تعيين_خاصية(ن، "س"، 10)
اطبع(هل_خاصية(ن، "ص"))     # صح
اطبع(هل_خاصية(ن، "لا"))    # خطأ
```
