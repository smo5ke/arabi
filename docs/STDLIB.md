## مكتبة `حساب` (Math)

```arabi
استورد حساب
```

### الدوال المثلثية

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `جيب(س)` | جيب الزاوية (بالدرجات) | `حساب.جيب(30)` = 0.5 |
| `تجيب(س)` | جيب التمام | `حساب.تجيب(0)` = 1.0 |
| `ظل(س)` | الظل | `حساب.ظل(45)` = 1.0 |
| `جيب_عكسي(س)` | الجيب العكسي (بالدرجات) | `حساب.جيب_عكسي(0.5)` = 30 |
| `تجيب_عكسي(س)` | جيب التمام العكسي | `حساب.تجيب_عكسي(1)` = 0 |
| `ظل_عكسي(س)` | الظل العكسي | `حساب.ظل_عكسي(1)` = 45 |
| `قم_اكبر(س1، س2)` | atan2(y, x) | `حساب.قم_اكبر(1، 1)` = 45 |

### الجذور والقوى

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `مربع(س)` | الجذر التربيعي | `حساب.مربع(144)` = 12 |
| `مكعب(س)` | الجذر التكعيبي | `حساب.مكعب(27)` = 3 |
| `قوة(أساس، أُس)` | الأُس | `حساب.قوة(2، 10)` = 1024 |

### التقريب والتحويل

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `قيمة_مطلقة(س)` | القيمة المطلقة | `حساب._abs(-5)` = 5 |
| `أرضية(س)` | التقريب للأسفل | `حساب.أرضية(3.7)` = 3.0 |
| `سقف(س)` | التقريب للأعلى | `حساب.سقف(3.2)` = 4.0 |
| `تقريب(س، خانات)` | التقريب | `حساب.تقريب(3.14159، 2)` = 3.14 |
| `حد_اعلى(س)` | إزالة الجزء العشري | `حساب.حد_اعلى(3.7)` = 3.0 |
| `راديان(درجة)` | تحويل لراديان | `حساب.راديان(180)` = 3.14159 |
| `درجة(راديان)` | تحويل لدرجة | `حساب.درجة(3.14159)` = 180 |

### اللوغاريتمات

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `لوغ(س)` | اللوغاريتم الطبيعي | `حساب.لوغ(2.718)` ≈ 1.0 |
| `لوغ10(س)` | اللوغاريتم العشري | `حساب.لوغ10(100)` = 2 |
| `لوغ2(س)` | اللوغاريتم الثنائي | `حساب.لوغ2(8)` = 3 |

### ثوابت

| الثابت | القيمة |
|--------|--------|
| `ط` | π ≈ 3.14159 |
| `ه` | e ≈ 2.71828 |
| `ن` | τ ≈ 6.28318 |

### إحصاءات ومسافات

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `متوسط(ق1، ق2، ...)` | متوسط الأعداد | `حساب.متوسط(1، 2، 3)` = 2.0 |
| `مسافة(ن1، ن2)` | المسافة الإقليدية | `حساب.مسافة([0، 0]، [3، 4])` = 5 |

### مثال شامل

```arabi
من حساب استورد ط، قوة، جذر، تقريب، قيمة_مطلقة

اطبع("ط:", ط)                     # 3.14159...
اطبع("2^10:", قوة(2، 10))         # 1024
اطبع("جذر 144:", جذر(144))        # 12
اطبع("تقريب:", تقريب(3.14159، 2)) # 3.14
اطبع("مطلق:", قيمة_مطلقة(-5))     # 5
```

## مكتبة `وقت` (Time)

```arabi
استورد وقت
```

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `الآن()` | ثواني منذ UNIX | `وقت.الآن()` = 1234567890 |
| `مللي()` | ميلي ثانية | `وقت.مللي()` = 1234567890000 |
| `وقت()` | كائن زمني كامل | يُرجع مترابط `{ثواني: 1234567890، ميلي: 0}` |
| `تاريخ()` | تاريخ اليوم | يُرجع مترابط `{يوم: 7، شهر: 6، سنة: 2026}` |
| `نَم(ثواني)` | نوم | `وقت.نَم(1)` — ينتظر ثانية |
| `_clock()` | عدّاد أداء | `وقت._clock()` — للقياس الدقيق |

### مثال

```arabi
استورد وقت

بداية = وقت._clock()
وقت.نَم(1)
نهاية = uptime._clock()

اطبع("الوقت:", نهاية - بداية)
```

## مكتبة `نص` (String)

```arabi
استورد نص
```

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `طول(ص)` | طول النص | `نص.طول("مرحبا")` = 5 |
| `استبدل(ص، قديم، جديد)` | استبدال | `نص.استبدل("مرحبا أحمد"، "أحمد"، "سارة")` |
| `افصل(ص، فاصلة)` | تقسيم | `نص.افصل("أ، ب، ج"، "، ")` |
| `ادمج(ق، فاصلة)` | دمج | `نص.ادمج(["أ"، "ب"]، " - ")` |
| `يحتوي(ص، جزء)` | فحص الاحتواء | `نص.يحتوي("مرحبا"، "رح")` |
| `ابحث(ص، نمط)` | البحث عن مؤشر | `نص.ابحث("مرحبا"، "رح")` = 1 |
| `يبدأ_ب(ص، نمط)` | يبدأ بـ | `نص.يبدأ_ب("مرحبا"، "مر")` |
| `ينتهي_ب(ص، نمط)` | ينتهي بـ | `نص.ينتهي_ب("مرحبا"، "حا")` |
| `شرعي(ص)` | هل هو عدد؟ | `نص.شرعي("123")` = صح |
| `شرعي_عشري(ص)` | هل هو عشري؟ | `نص.شرعي_عشري("3.14")` = صح |
| `أحرف_كبيرة(ص)` | تحويل لأحرف كبيرة | `نص.أحرف_كبيرة("مرحبا")` = "مرحبا" |
| `أحرف_صغيرة(ص)` | تحويل لأحرف صغيرة | `نص.أحرف_صغيرة("مرحبا")` = "مرحبا" |
| `انزف(ص)` | إزالة المسافات | `نص.انزف("  مرحبا  ")` = "مرحبا" |
| `انزف_البداية(ص)` | إزالة من البداية | |
| `انزف_النهاية(ص)` | إزالة من النهاية | |
| `مطابق(ص، نمط)` | تطابق Regex | `نص.مطابق("123"، "\\d+")` |

### مثال

```arabi
استورد نص

ص = "مرحبا بالعالم"
اطبع(طول(ص))                          # 13
اطبع(ص.استبدل("العالم"، "عربي"))    # مرحبا عربي
اطبع(ص.اقسم(" "))                     # [مرحبا, بالعالم]
```

## مكتبة `عشوائي` (Random)

```arabi
استورد عشوائي
```

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `عشوائي()` | رقم عشوائي 0-1 | `عشوائي.عشوائي()` = 0.723... |
| ` صحيح_عشوائي(حد_أسفل، حد_أعلى)` | رقم صحيح عشوائي | `عشوائي. صحيح_عشوائي(1، 100)` = 42 |

```arabi
    اطبع(عشوائي. صحيح_عشوائي(1، 6))    # رمي نرد
```

## مكتبة `كائن` (JSON)

```arabi
استورد كائن
```

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `تحويل(قائمة)` | تحويل لنص JSON | `كائن.تحويل({"أ": 1})` |
| `تحليل(نص)` | تحليل نص JSON | `كائن.تحليل('{"أ": 1}')` |

### مثال

```arabi
استورد كائن

بيانات = {"اسم": "عربي"، "إصدار": 2}
نص = كائن.تحويل(بيانات)
اطبع(نص)    # {اسم: عربي, إصدار: 2}

محلل = كائن.تحليل('{"أ": 1، "ب": 2}')
اطبع(محلل["أ"])    # 1
```

## مكتبة `نظام` (OS)

```arabi
استورد نظام
```

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `اسم_النظام()` | اسم نظام التشغيل | `نظام.اسم_النظام()` = "windows" |
| `وجود(مسار)` | فحص وجود ملف | `نظام.وجود("ملفي.txt")` = صح |
| `اقرأ_ملف(مسار)` | قراءة ملف كامل | `نظام.اقرأ_ملف("ملفي.txt")` |
| `اكتب_ملف(مسار، محتوى)` | كتابة ملف | `نظام.اكتب_ملف("ملفي.txt"، "مرحبا")` |
| `قائمة_ادارة(مسار)` | قائمة المجلدات | `نظام.قائمة_ادارة(".")` |
| `ادارة(مسار)` | إنشاء مجلد | `نظام.ادارة("مجلد_جديد")` |

### مثال

```arabi
استورد نظام

اطبع("النظام:", نظام.اسم_النظام())
اذا نظام.وجود("ملفي.txt"):
    محتوى = نظام.اقرأ_ملف("ملفي.txt")
    اطبع(محتوى)
```

## مكتبة `اختبار` (Test)

```arabi
استورد اختبار
```

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `طالما(شرط، رسالة)` | تحقق من شرط | `اختبار.طالما(س > 0، "يجب أن يكون موجب")` |
| `طالما_يساوي(أ، ب، رسالة)` | تحقق من التساوي | `اختبار.طالما_يساوي(5 + 5، 10)` |
| `طالما_ليسا(أ، ب، رسالة)` | تحقق من عدم التساوي | `اختبار.طالما_ليسا(5، 10)` |

### مثال

```arabi
استورد اختبار

س = 10
اختبار.طالما(س > 0، "يجب أن يكون موجب")
اختبار.طالما_يساوي(س، 10، "القيمة يجب أن تكون 10")
اختبار.طالما_ليسا(س، 0، "لا يمكن أن يكون صفر")
```

## المكتبات المخصصة

يمكنك إنشاء حزمك الخاصة:

### هيكل الحزمة

```
مكتبتي/
├── __بداية__.txt    # ملف التهيئة (يُنفّذ عند الاستيراد)
├── وحدة1.txt       # وحدة فرعية
└── وحدة2.txt       # وحدة فرعية
```

### الاستيراد

```arabi
# استيراد وحدة
استورد مكتبتي.وحدة1

# استيراد أسماء محددة
من مكتبتي.وحدة1 استورد شيء1، شيء2

# استيراد مباشر من الحزمة
من مكتبتي استورد شيء
```

### مثال على ملف التهيئة

```arabi
# مكتبتي/__بداية__.txt
اثن_على("تم تحميل الحزمة!")
س = 0
```
