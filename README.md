# عربي (Arabi) - لغة برمجة عربية

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Rust](https://img.shields.io/badge/Rust-1.70+-orange.svg)](https://www.rust-lang.org)
[![Tests](https://img.shields.io/badge/tests-129%20passed-brightgreen.svg)]()

لغة برمجة عربية كاملة ببنية Python-like و اداء C++-competitive.

## الميزات الرئيسية

- **بنية عربية بالكامل**: متغيرات، دوال، اصناف، استيراد — كلها باسماء عربية
- **39 كلمة مفتاحية**: جميع ميزات Python تقريباً
- **اداء عالي**: VM مبني بـ Rust مع تحسينات متقدمة
- **170+ دالة مدمجة**: رياضيات، نصوص، وقت، عشوائي، ملفات، JSON، نظام
- **نظام اصناف كامل**: وراثة، طرق، magic methods، decorators
- **التغليف (Closures)**: دوال داخلية مع `محلي` و `عام`
- **معالجة اخطاء**: `حاول`/`خلل`/`اخيرا` مع استثناءات مخصصة وارقام اسطر
- **مدير سياق**: `باستخدام` مع `__ادخل__`/`__اترك__`
- **استيراد و حزم**: 6 وحدات مدمجة + ملفات + نظام حزم
- **المولّدات**: `yield`، `yield from`، `ابعث()` (send)
- **CLI متقدم**: REPL تفاعلي + ادارة حزم + الوان

## المتطلبات

- [Rust](https://www.rust-lang.org/tools/install) 1.70 او احدث

## التثبيت

```bash
git clone https://github.com/arabi-lang/arabi.git
cd arabi
cargo build --release
```

## الاستخدام

```bash
# تشغيل ملف
arabi ملفي.txt

# REPL تفاعلي
arabi

# وضع التصحيح (يعرض bytecode)
arabi -d ملفي.txt

# اوامر الحزم
arabi pakg create حزمة_جديدة    # انشاء حزمة
arabi pakg list                  # عرض الحزم
arabi pakg info حزمة             # معلومات حزمة
arabi pakg remove حزمة           # حذف حزمة
arabi pakg update حزمة           # تحديث حزمة
```

## نظام الحزم

```bash
# انشاء حزمة جديدة
arabi pakg create حسب

# هذا ينشئ:
#   __pakgs__/حسب/__beginning__.txt
#   __pakgs__/حسب/pakg.json

# استيراد الحزمة في الكود
# استورد حسب
```

## REPL التفاعلي

```bash
# تشغيل REPL
arabi

# اوامر REPL
/مساعدة        # عرض المساعدة
/خروج          # الخروج
/حالة          # عرض المتغيرات

# امثلة في REPL
عربي> س = 10
عربي> اطبع(س + 5)
15
```

## امثلة سريعة

### المتغيرات والعمليات
```
س = 10
ص = "مرحبا"
اطبع(س + 5)        # 15
اطبع(س ^ 2)        # 100
اطبع("مرحبا " + "عالم")  # مرحبا عالم
```

### الدوال
```
دالة جمع(ا، ب):
    ارجع ا + ب

اطبع(جمع(3، 4))    # 7

# *args و ^kwargs
دالة مختلطة(ا، *الاضافي، ^الكثرة):
    اطبع("ا:", ا، "اضافي:", الاضافي، "كثرة:", الكثرة)
```

### الاصناف
```
صنف مخروط:
    دالة __تهيئة__(هذا، نق = 1):
        هذا.نق = نق

    دالة مساحة(هذا):
        ارجع هذا.نق ^ 2 * 3.14

م = مخروط(5)
اطبع(م.مساحة())    # 78.5...
```

### المولّدات
```
دالة عداد(حد):
    س = 0
    بينما س < حد:
        سلم(س)
        س += 1

لكل ع في عداد(5):
    اطبع(ع)    # 0, 1, 2, 3, 4
```

### معالجة الاخطاء
```
حاول:
    س = 10 / 0
خلل قسمة_بصفر:
    اطبع("خطا!")
نهاية:
    اطبع("تنظيف")
```

### تركيبات
```
مربعات = [س ^ 2 لكل س في مدى(6)]
اطبع(مربعات)    # [0, 1, 4, 9, 16, 25]

ف = {س: س ^ 2 لكل س في مدى(5)}
اطبع(ف)        # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
```

| Nested Loops (1M) | ~0.045s | ~0.135s | **~3x اسرع** |
| Fibonacci (recursive 35) | ~0.003s | ~0.019s | **~6x اسرع** |
| Closures (100K) | ~0.004s | ~0.005s | **~1.3x اسرع** |

### تحسينات الاداء المطبقة

| التحسين | الوصف |
|---------|-------|
| **JIT Compilation** | Cranelift JIT للفيبوناتشي والحلقات |
| **String Interning** | بحث O(1) للاسماء بدلاً من O(n) |
| **Peephole Optimizer** | 30+ تمرير تحسين bytecode |
| **Fused Opcodes** | دمج العمليات في opcode واحد |
| **INT_INT Specialization** | تخصيص عمليات الاعداد الصحيحة |
| **FastMethod Dispatch** | تمرير مباشر للطرق البسيطة |
| **Unsafe Field Access** | تخطي RefCell للقراءات |
| **Inline Fields** | تخزين حتى 4 حقول داخل الكائن |
| **mimalloc** | خاضع الذاكرة عالي الاداء |
| **Instance-first Fast Path** | مسار سريع لانشاء الكائنات |
